Introduction

What is ML in Video Games?

Machine learning is a branch of artificial intelligence where computers learn patterns from data without being explicitly programmed to do so. In the video game industry, this has become extremely important because enormous amounts of data are generated.

Companies like Valve, EA, Riot Games, and Nintendo use machine learning to improve matchmaking, predict sales, detect cheating, and recommend games to users.

This project showcases 4 concrete use cases of machine learning in the gaming world.

Types

Types of Learning

Classification of ML algorithms

S
Supervised

The model learns from labeled data. Example: game data where we already know if they sold well or poorly.

Used in this project
N
Unsupervised

No labels. The algorithm finds groupings on its own. In gaming, it groups players by behavior or finds similar communities.

R
Reinforcement

Learns through trial and error with rewards. Widely used to create intelligent NPCs and bots that learn on their own.
