> Que es ML en videojuegos?_

El machine learning es una rama de la inteligencia artificial donde las computadoras aprenden patrones a partir de datos sin que uno las programe directamente para eso. En la industria de los videojuegos esto se ha vuelto super importante porque se generan cantidades enormes de datos: estadisticas de jugadores, ventas, comportamiento en el juego, reviews, etc.

Empresas como Valve, EA, Riot Games y Nintendo usan machine learning para mejorar el matchmaking, predecir ventas, detectar trampas y recomendar juegos a los usuarios.

En este trabajo se muestran 2 aplicaciones concretas de machine learning en el mundo gaming usando clasificacion supervisada.

> SELECT LEARNING TYPE_

[ SUPERVISADO ]

El modelo aprende con datos etiquetados. Ejemplo: datos de juegos donde ya sabemos si vendieron bien o mal. Es el que usamos en este trabajo.

[ NO SUPERVISADO ]

No hay etiquetas. El algoritmo encuentra agrupaciones solo. En gaming agrupa jugadores por comportamiento o encuentra comunidades similares.

[ POR REFUERZO ]

Aprende por prueba y error con recompensas. Muy usado para crear NPCs inteligentes y bots que aprenden solos (como el bot de OpenAI en Dota 2).
