Predice la nota estimada de un estudiante basandose en las horas de estudio

Concepto

Como funciona?

La regresion lineal busca la relacion entre una variable independiente (horas de estudio) y una variable dependiente (nota obtenida).

El modelo ajusta una linea recta y = mx + b que minimiza el error entre las predicciones y los valores reales.

Es uno de los algoritmos mas basicos de machine learning y sirve como base para entender modelos mas complejos.
