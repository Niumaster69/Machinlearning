Unsupervised Learning — Basic Concepts

Finding Hidden
Patterns in Data

Understanding unsupervised learning, clustering, and the K-Means algorithm — the tools that let machines discover structure in data without any labels.

Definition

What is Unsupervised Learning?

Unsupervised learning is a branch of machine learning where the algorithm receives only input data, without any labels or expected outputs. Instead of being told what the correct answer is, the model must discover the underlying structure of the data on its own.

This contrasts with supervised learning — where algorithms such as Linear Regression, Logistic Regression, and KNN learn from labeled examples (games with known ratings, games with known sales, etc.).

In the video game industry, unsupervised learning is used to segment players, group similar games, detect anomalies such as cheaters or bots, and compress data for recommendation systems.

Supervised
Learns from labels

Input → Output pairs. The algorithm minimizes the prediction error against known answers.

Unsupervised
Learns from structure

Input only. The algorithm finds groups, patterns, or compressed representations by itself.

Core Task

What is Clustering?

Clustering is the most common unsupervised learning task. Its goal is to partition a dataset into groups (called clusters) such that items within the same group are more similar to each other than to items in other groups.

Similarity is measured with a distance metric (usually Euclidean distance). Points that are close together in the feature space get assigned to the same cluster; points that are far apart end up in different clusters.

There is no ground truth to compare against — the algorithm defines what a "good" clustering looks like through a mathematical criterion (typically minimizing the variance inside each cluster).

Algorithm

What is K-Means?

K-Means is the most widely used clustering algorithm. It partitions a dataset into K clusters, where K is chosen by the user before training. Each cluster is represented by its centroid — the mean of all points assigned to that cluster.

The algorithm iteratively refines the position of the centroids and the assignments of the points until the clustering stops changing (convergence) or a maximum number of iterations is reached.

K-Means is simple, fast, and scales well to large datasets, which is why it is the default choice for most segmentation problems.

Building Blocks

Key Concepts

The main ideas behind the K-Means algorithm

Concept
Centroids

A centroid is the geometric center of a cluster — the arithmetic mean of all the points that belong to it. In a 2-feature problem, a centroid is a point (x, y); with more features, it is a vector of means.

Concept
Distance Metric

K-Means uses Euclidean distance to measure how far a data point is from each centroid. The point is then assigned to the cluster whose centroid is closest.

Concept
Variance (WCSS)

The Within-Cluster Sum of Squares is the sum of squared distances from each point to its assigned centroid. K-Means aims to minimize this quantity — smaller WCSS means tighter, more compact clusters.

Concept
Convergence

The algorithm has converged when the centroids stop moving (or move very little) between iterations. At that point, further updates no longer reduce the variance in a meaningful way.

Math

Euclidean Distance Formula

The distance between a data point P = (x1, y1) and a centroid C = (x2, y2) is:

d(P, C) = √( (x1 − x2)² + (y1 − y2)² )
Euclidean Distance — Straight-line distance in 2D space

For higher dimensions, the formula extends naturally by adding one squared difference per feature before taking the square root.

WCSS = Σ Σ d(xi, ck)²
Within-Cluster Sum of Squares (the value K-Means minimizes)
How it works

The Iterative Process

K-Means refines its solution through a loop of four steps

1
Initialize Centroids

Choose K initial centroid positions. They can be random, picked manually, or selected by a smarter strategy like K-Means++. In our manual exercise we pick them by hand, spread across the data range.

2
Assign Points to the Nearest Centroid

For every data point, compute the Euclidean distance to each of the K centroids. Assign the point to the cluster whose centroid is the closest.

3
Recalculate Centroids

Once every point has been assigned, the new centroid of each cluster is the mean of all points inside it. Centroids "move" toward the center of their group.

4
Repeat Until Convergence

Go back to step 2 with the updated centroids. Repeat until centroid positions stop changing or until a maximum number of iterations is reached. The total variance (WCSS) decreases on every iteration.

Behavior

Variance and Convergence

At each iteration, K-Means decreases (or keeps equal) the value of the WCSS. Points are reassigned to closer centroids, and centroids move to the average position of their new members — both operations can only reduce the overall variance.

When the algorithm reaches a point where no point changes cluster between iterations, the centroids also stop moving. This is called convergence. K-Means is guaranteed to converge in a finite number of steps, though it may reach a local minimum rather than the global optimum.

A good clustering is one where WCSS is small (tight clusters) and the clusters are well separated from each other. Plotting WCSS across iterations produces a monotonically decreasing curve that flattens near the converged value.

Applications

Real-World Applications

Where unsupervised learning makes an impact

Gaming
Player Segmentation

Studios group players by behavior (casual, competitive, whales, explorers) to design balanced matchmaking and tailored content.

E-commerce
Customer Segmentation

Retailers cluster customers by purchase history to target promotions, recommend products, and estimate lifetime value.

Security
Anomaly Detection

Points that do not fit into any cluster (or fall into tiny clusters) are often outliers — useful for detecting fraud, cheats, and intrusions.

Healthcare
Patient Subtypes

Clustering identifies groups of patients with similar symptoms, enabling personalized treatment plans and clinical research stratification.

Marketing
Market Basket Analysis

Clustering products by co-purchase patterns reveals natural groupings of items, driving store layout and cross-sell strategy.

Vision
Image Compression

K-Means on pixel colors reduces an image's palette to K representative colors, shrinking file size with minimal perceptual loss.

Trade-offs

Advantages and Limitations

Advantages
  • Simple to understand and implement
  • Fast and scales well to large datasets
  • Works with any number of features
  • Gives interpretable centroids as group prototypes
Limitations
  • K (number of clusters) must be chosen beforehand
  • Sensitive to the initial centroid positions
  • Assumes spherical, similarly-sized clusters
  • Affected by feature scale — requires standardization
  • May converge to a local minimum (not global)
