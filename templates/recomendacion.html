Sistema de recomendacion con KNN basado en gustos del jugador

> Descripcion del problema_

Hoy hay miles de videojuegos en Steam, PlayStation Store o Nintendo eShop. Es dificil encontrar juegos nuevos que te gusten entre tanta oferta. Por eso las tiendas usan sistemas de recomendacion.

La idea: si a dos jugadores les gustan los mismos juegos, un juego que le gusta a uno probablemente le guste al otro. Esto es filtrado colaborativo, lo que usan Netflix, Spotify y Steam.

Se simulo un dataset de 5,000 jugadores con horas por genero, ratings y edad. Se usa KNN para encontrar jugadores similares y recomendar generos.

> WEAPON: K-Nearest Neighbors_

KNN es de los algoritmos mas simples. Para clasificar un dato nuevo busca los K mas parecidos (vecinos cercanos) y ve a que clase pertenece la mayoria.

Ejemplo: K=7, se buscan los 7 jugadores mas parecidos a ti. Si 5 de 7 prefieren RPG, se te recomienda RPG.

[ BUFFS ]
  • Muy facil de entender
  • No necesita entrenamiento (lazy learner)
  • Funciona bien con datos agrupados
  • Se adapta facil a datos nuevos
[ DEBUFFS ]
  • Lento con datasets muy grandes
  • Sensible a la escala (normalizar)
  • No va bien con muchas dimensiones
  • Elegir K es trial and error

> SOURCE CODE_

# Importar librerias
import numpy as np
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler
from sklearn.neighbors import KNeighborsClassifier
from sklearn.metrics import accuracy_score, classification_report

# Simular dataset de 5000 jugadores
np.random.seed(42)
n = 5000

data = {
    'horas_action': np.random.randint(0, 500, n),
    'horas_rpg': np.random.randint(0, 500, n),
    'horas_sports': np.random.randint(0, 500, n),
    'horas_shooter': np.random.randint(0, 500, n),
    'rating_promedio': np.random.uniform(1.0, 5.0, n).round(1),
    'edad': np.random.randint(13, 46, n),
}
df = pd.DataFrame(data)

# Genero favorito = el que tiene mas horas
cols_horas = ['horas_action', 'horas_rpg', 'horas_sports', 'horas_shooter']
df['genero_favorito'] = df[cols_horas].idxmax(axis=1).map({
    'horas_action': 'Action',
    'horas_rpg': 'RPG',
    'horas_sports': 'Sports',
    'horas_shooter': 'Shooter'
})

# Separar X e y
X = df.drop('genero_favorito', axis=1)
y = df['genero_favorito']

# Normalizar
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# Dividir datos
X_train, X_test, y_train, y_test = train_test_split(
    X_scaled, y, test_size=0.2, random_state=42)

# Entrenar KNN con K=7
knn = KNeighborsClassifier(n_neighbors=7)
knn.fit(X_train, y_train)

# Evaluar
pred = knn.predict(X_test)
print(f"Accuracy: {accuracy_score(y_test, pred)}")
print(classification_report(y_test, pred))
